# Daily Journal Templates

Complete template reference for the Daily Journal skill.

## Daily Journal (Default)

Standard template for normal days.

```markdown
# {{date:YYYY-MM-DD dddd}}

## 🌅 早晨设定
- 今天的状态：[1-10分]
- 今天想focus on的一件事：
- 如果今天结束时我会感到满意，是因为：

---

## 📝 今日记录

### 发生了什么？
- 今天做了什么事？（至少3件）
- 和谁有互动？
- 身体/情绪状态如何？

### 为什么会这样？
- 选择做这些事的原因是什么？
- 今天的状态和昨天/上周比有什么不同？
- 有什么重复出现的模式吗？

### 这说明了什么？
- 如果要给今天打个标签，是什么？（如：恢复日、推进日、混乱日）
- 今天的选择反映了我现在在意什么？
- 有什么是我想改变/保持的？

---
```

## Day Interrupt Check-in

Template for interrupt prompts (stored separately, not in main journal).

```markdown
# 🔔 Day Interrupt Check-in

> 用于日间随机提醒，打破自动驾驶模式
> 建议：每天设置3个随机时间的提醒

---

## ⏰ Check-in Time: {{time}}

### 现在在做什么？
-

### 快速反思（选一个问题回答）

**上午时段：**
- 现在在做的事，是"想成为的我"会做的吗？
- 我在逃避什么？

**下午时段：**
- 过去2小时，我在用什么"合理借口"？
- 如果有人拍下过去2小时，会得出我想要什么样的生活？

**晚上时段：**
- 今天最接近"理想自我"的时刻是？
- 今天最远离的时刻是？
- 我在向想要的生活靠近，还是在向不想要的生活靠近？

---
```

## Template Selection Rules

| Condition | Template |
|-----------|----------|
| Energy ≤4 | Low Energy |
| Mentions parenting/kids | Parenting Day |
| Mentions confusion/struggle | Reflection |
| Mentions completion/finish | Project Day |
| Default | Daily Journal |

## Tags Reference

| Tag | Use When |
|-----|----------|
| #insight | 对自己/生活/工作的感悟 |
| #idea/content | 可能写成内容的想法 |
| #idea/product | 产品/项目想法 |
| #seed/story | 可以讲的故事 |
| #seed/observation | 有趣的观察 |
| #fleeting | 需要进一步思考的闪念 |
| #method | 学到的方法/流程 |
| #principle | 总结的原则 |
| #case | 案例/经历 |
| #question | 待探索的问题 |
